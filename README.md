# Qt实现WAV音频文件的波形图与频谱图显示

本项目提供了一个使用Qt框架实现的WAV音频文件波形图与频谱图显示的资源文件。通过该资源文件，您可以轻松地在Qt应用程序中加载并显示WAV音频文件的波形图和频谱图。

## 使用说明

1. **下载资源文件**: 下载并解压`Qt显示WAV音频文件的波形图频谱图.rar`文件。
2. **导入项目**: 将解压后的文件导入到您的Qt项目中。
